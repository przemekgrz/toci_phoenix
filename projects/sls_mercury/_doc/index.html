Fixed Header

To activate fixed header, you just need to add "data-fixed" attribute near to "met_main_nav" in HTML pages.

Boxed Layout

To activate boxed layout, you just need to add "met_boxed_layout" class near to "met_page_wrapper" in HTML pages.

Retina Usage

Please see: http://retinajs.com/

Custom Scrollbar

To deactivate custom scrollbar you need to make ' data-smooth-scrolling="1" ' to ' data-smooth-scrolling="0" '.

Color Changing

Go to demo page: http://metcreative.com/demo_viewer/index.php?theme=cacoon

Choose the color that you wish from Style Switcher which is at left side of your screen.

Show codes and copy them.

In your HTML pages you will find ' <style id="colorChanges" type="text/css"> ', put the codes in this tag.

And your theme has a new color.

Contact Map

First you need lat and lng values of your place from Google Maps. This website can help you: http://www.doogal.co.uk/LatLong.php

When you get those 2 values, go to js/custom.js and search for ' Gmaps JS for Google Maps ', there you will see options for your contact map location.

Twitter Feed

This process is straightforward and you should have a set of keys within a few minutes.

  1. Visit https://dev.twitter.com/apps/ and sign in using your Twitter username and password. This doesn’t have to be the username or password of the stream you need access to, just a Twitter account you control.
  2. Select ‘Create new application’ and enter the application details.
    1. The name and description can be anything you like really, but you can’t use ‘Twitter’ in the name.
    2. The website field can be your main website and doesn’t have to be the site where your Twitter feed or feeds are located.
    3. Callback URL can be left blank
  3. Enter the CAPTCHA info and click create
  4. On the next details screen, click ‘create my access token’. You may need to refresh the page after a few seconds if it doesn’t appear automatically.
  5. Make a note of the Consumer key, Consumer secret, Access token and Access token secret as highlighted below.

 

 

Now you have your keys. Go to twitter/get-tweets1.1.php. There you will see some variables with keys etc. Put your own keys in there. And the $twitteruser variable is the username of the feed owner.

 

Now you should be able to pull latest tweets automatically.
